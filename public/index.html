FAQ

Clear answers before launch

How long are invitation files stored?

All project data, Cloudinary assets, generated folders, and ZIP files expire after 24 hours.

Can the website work after download?

Yes. The ZIP contains standalone HTML, CSS, JavaScript, and data files for the generated invitation.

Is Cloudinary upload handled by the browser?

Yes. The dashboard requests a signed upload signature, then sends valid images directly to Cloudinary.

Do I need Bootstrap or Tailwind?

No. WeddingCraft is built with HTML5, CSS3, and vanilla JavaScript.
